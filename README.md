# Handcrafted
Factorio mod which adds a toolbox and consumable tools to speed up your manual crafting. Extends original [Handcrafted v3.0.0 mod](https://mods.factorio.com/mod/handcrafted) by [Kubius](https://mods.factorio.com/user/Kubius)

## Content
### Toolbox
![Hand tools](/graphics/toolbox.png)<br>
Base container for tools. Must be equipped to use the tool.
### Hand tools
![Hand tools](/graphics/hand-tools.png)<br>
* Increase manual crafting speed by 75%;
* Durability: 1000 sec.
### Pneumatic tools
![Hand tools](/graphics/pneumatic-tools.png)<br>
* Increase manual crafting speed by 90%;
* Durability: 800 sec.
### Cordless tools
![Hand tools](/graphics/cordless-tools.png)<br>
* Increase manual crafting speed by 95%;
* Durability: 400 sec.
### Electric tools
![Hand tools](/graphics/electric-tools.png)<br>
* Increase manual crafting speed by 100%;
* Durability: 800 sec.
#### P.s. The tool's stats have been intentionally increased. Choose the most fair percentages for your gameplay.
<br>

## Changelog
### Version 0.3.1
* Fixed crafting handling: crafting no longer breaks when opening the map (at the cost of a microscopic performance decrease);
* Integrated tools into the research tree in a lore-friendly way;
* Added new types of tools (including sprites); rebalanced stats for all tool tiers to provide smoother and more engaging progression;
* Updated crafting recipes to better reflect real-world tool construction and to allow reuse of lower-tier tools;
* Added Russian localization.
### Version 0.3.0
* Factorio 2.0 compatibility.
### Version 0.2.0
* Added art for basic and power hand tools;
* Dramatically increased the base lifespan of hand tools (basic 30 > 160, power 60 > 240) to make them more practical for minor component fabrication.
### Version 0.1.0
* Initial release (Kubius).
